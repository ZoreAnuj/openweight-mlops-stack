# OpenWeight MLOps Stack

A production-grade open-source MLOps pipeline for developing and deploying open-weight models. This project explores building enterprise-ready infrastructure to support the full lifecycle of models that aim to rival proprietary solutions.

## Key Features
*   Complete pipeline from ETL and model training to FastAPI deployment.
*   Orchestration with Apache Airflow and experiment tracking via MLflow.
*   Integrated CI/CD practices for automated testing and deployment.
*   Includes sample datasets and configurations for rapid prototyping.

## Tech Stack
Python, FastAPI, Apache Airflow, MLflow, Scikit-learn, Docker, GitHub Actions

## Getting Started
1.  Clone the repo: `git clone https://github.com/zoreanuj/openweight-mlops-stack.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the services: `docker-compose up`
